Instead of using GWAS data from our database, you are welcome to upload your own GWAS data. Your file must* be tab separated (either .tsv or .txt), and have the following columns: Study ID?, Trait?, RsID?, Chromosome?, Position?, Risk Allele?, Odds Ratio?, P-value?, and Super Population?. Optional column headers that will be included if present are: Citation?, P-Value Annotation?, and Reported Trait?. All other columns in the file will be ignored. If more than one odds ratio exists for an rsID within a trait/study combination, the tool will throw an error and cease calculating.
