How to Input List of RS IDs

RS ids can be input with or without a genotype. If an rs id is input without a genotype, all risk alleles at that position will be used in calculating the polygenic risk scores and in the subsequent risk score matrices. If genotypes are available, each allele must be specified on a separate line. RS IDs and genotypes are separated by a colon. Separate RS IDs by either a comma, space, or an end-of-line character, as shown below.
