How to Use the Calculator

You can calculate a polygenic risk score by inputting rs IDs in the box below or uploading a VCF file. RS ids can be input with or without a genotype. If an rs ID is input without a genotype, all risk alleles at that position will be used in calculating the polygenic risk scores. If genotypes are available, alleles can be indicated after the rs ID. One or two alleles may be used. When only one allele is specified, the other will be assumed to be the risk allele. RS IDs and genotypes are separated by a colon. Separate rs IDs by either a comma, space, or an end-of-line character, as shown below.

Once you have input the rs IDs or uploaded a VCF file, select the disease you would like to test. The corresponding genome-wide association (GWA) study you select will contain the variants and odd ratios used to calculate the score. The p-value cutoff you select will decide which variants from the GWA study pass the significance threshold for calculating the score. Finally, select the reference genome from which your rs IDs or VCF file were determined.
